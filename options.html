ABOUT

This extension filters DW pages such that the only comments that show up are the ones specifically whitelisted. Think of it as an advanced Ctrl + F.

CURRENTLY FILTERING TO

You can add multiple terms, just split them up with commas (,).

Wildcard input guide

By default, terms that are left as is will be searched for as is.

ota will show results that only have ota, not words that start with ota such as otaku or contain ota such as rotation.

To include more, use the wildcard asterisk (*).

e.g. fate/* will include fate/extra, fate/zero, fate/apocrypha, fate/ series, etc.
